# Welcome to Ghana Exchange Rates
The Ghana Exchange Rate is a Python package to ease checking of currency exchange rates. It is a free software, distributed under the BSD license, 
and available on PyPI. The documentation is available on Read the Docs and development takes place on GitHub.
For full documentation visit [mkdocs.org](https://www.mkdocs.org).

## Commands

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.
